🌌 Nova UI — Premium CSS Design System
Nova UI is a curated collection of high-end, responsive UI components built with Vanilla CSS and React. It is designed for developers who want to create stunning "dev-tool" interfaces with a minimalist, high-contrast "True Black" aesthetic.


✨ Features
True Black Aesthetic: Optimised for OLED displays with deep blacks and vibrant neon accents.
Micro-Animations: Custom-crafted CSS animations including Cyberpunk glitches, 3D push effects, and shimmering gradients.
Zero Dependencies: Core styles are written in pure CSS, making them ultra-lightweight and easy to port.
Copy-Paste Ready: Each component comes with its own isolated CSS and HTML structure for instant integration.
Glassmorphism & Neumorphism: Modern design trends baked into the core components.
🛠️ Tech Stack
Core: React 18 & Vite
Styling: Vanilla CSS (Custom Tokens & Utilities)
Icons: Lucide React
Typography: Inter / Outfit (Modern Sans)
🚀 Getting Started
Installation
Clone the repository:

bash
git clone https://github.com/ShivamChaturvedi54/css-ui-library.git
Install dependencies:

bash
npm install
Start the development server:

bash
npm run dev

🎨 Design Philosophy
Nova UI follows a "Functionally Aesthetic" philosophy. We believe that developer tools shouldn't just be functional—they should look premium. We focus on:

High Contrast: White-on-Black clarity.
Motion with Purpose: Using GLITCH and SHIMMER to guide user focus.
Modular CSS: Keeping styles encapsulated within components.
